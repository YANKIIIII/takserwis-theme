# Tak Serwis - WordPress Theme 🚗🔧

Custom WordPress theme developed for **Tak Serwis** – a professional auto repair and vehicle inspection station (Stacja Kontroli Pojazdów) located in Poznań, Poland.

## 🌟 Key Features

*   **Mobile-First Responsive Design:** Fully optimized for all devices with custom CSS breakpoints for service cards and a perfectly centered mobile burger menu.
*   **Multi-language Support:** Integrated with Polylang (`pll`) for seamless switching between Polish, Ukrainian, and Russian.
*   **Online Booking Integration:** Includes direct widget integration with `zarezerwuj.pl` for online customer appointments.
*   **Custom Post Types & Taxonomies:** Tailored architecture for managing services, auto parts, and customer reviews.
*   **Performance Optimized:** Cleaned of unnecessary artifacts, legacy code, and server bloat.

## 🛠️ Tech Stack
*   **CMS:** WordPress
*   **Styling:** CSS3, Bootstrap (customized overrides), SASS/Gulp workflows
*   **Scripts:** JavaScript, jQuery
*   **Integrations:** Google Tag Manager, Orły Motoryzacji API/Badges

## 📱 Mobile UI Fixes
Recent updates include strict CSS overrides in `style2.css` to prevent Bootstrap classes from breaking the mobile layout:
*   Neutralized absolute positioning bugs on service imagery.
*   Enforced structural centering for mobile navigation dropdowns.
